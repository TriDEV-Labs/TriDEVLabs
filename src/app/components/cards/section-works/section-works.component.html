Projeto Futebol Score

O Match Point Status é um projeto web dedicado ao acompanhamento de diversas ligas de futebol. Desenvolvido como uma demonstração das minhas habilidades em PHP, CSS e JavaScript, o site utiliza a API fornecida pelo .football-data.org para obter informações em tempo quase sobre partidas, pontuações e estatísticas de jogadores.

Maridão na cozinha

Desenvolvimento de um espaço online dedicado à culinária, com mais de 300 receitas catalogadas e organizadas.
Utilização de tecnologias como Angular e banco de dados NoSQL MongoDB para construir um site responsivo e acessível em diferentes dispositivos.
Compromisso em oferecer uma fonte confiável de inspiração culinária, com recursos como pesquisa avançada e dicas sobre técnicas de cozimento.
Empenho em proporcionar uma experiência completa aos usuários. Preparação para o lançamento em breve, com espaço para sugestões e ideias dos usuários.
